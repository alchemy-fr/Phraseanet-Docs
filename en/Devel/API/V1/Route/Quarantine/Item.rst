Quarantine item
===============

About
-----

Return an item of the quarantine given its Id

.. code-block:: bash

    /api/v1/quarantine/item/{quarantine_id}/

======================== =====
 Informations
======================== =====
 HTTP Method              GET
 Requires Authentication  Yes
======================== =====

Parameters
----------

======================== ============== =============
 Parameters               Type           Information
======================== ============== =============
 quarantine_id            integer        The id of the item
======================== ============== =============

Response Fields
---------------

============== ================================
 Field          Description
============== ================================
 item           The item
============== ================================
