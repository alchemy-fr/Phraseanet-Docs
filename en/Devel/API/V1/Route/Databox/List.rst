Databox List
============

About
-----

  Return available Databoxes

  .. code-block:: bash

    /api/v1/databoxes/list/

  ======================== =====
   Informations
  ======================== =====
   HTTP Method              GET
   Requires Authentication  Yes
  ======================== =====

Parameters
----------

  No parameters for this route

Response Fields
---------------

  ========== ================================
  Field       Description
  ========== ================================
  databoxes   The list of available databoxes
  ========== ================================
