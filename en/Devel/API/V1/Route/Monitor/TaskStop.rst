Stop a task
===========

About
-----

  Stop a task and return its status

  .. code-block:: bash

    /api/v1/monitor/task/{task_id}/stop/

  ======================== =====
   Informations
  ======================== =====
   HTTP Method              POST
   Requires Authentication  Yes
  ======================== =====

Parameters
----------

  ======================== ============== =============
   Parameters               Type           Information
  ======================== ============== =============
   task_id                  integer        The task id
  ======================== ============== =============

Response Fields
---------------

  ========== ================================
   Field      Description
  ========== ================================
   task       The task you're monitoring
  ========== ================================
