Documentation API v1
====================

Phraseanet API v1 gives you access to lots of Phraseanet data and lets
you interact with them.
Integrate the search engine, build your own applications and plugins!
Give us your feedback and if you publish your code, let us know and
we'll also do so.

* Read the ':doc:`overview <V1/Overview>`
* Get familiar with :doc:`oAuth2 <V1/Authentication>`
* Find your :doc:`route <V1/Routes>`!

If you ever find bugs, please report them on the `mailing-list`_ or create a
ticket on our `bug tracker`_.
