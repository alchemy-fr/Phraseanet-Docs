Basket Add Element
==================

About
-----

Adds a new element to an existing basket.

.. code-block:: bash

    /api/v2/baskets/{basketId}/records

======================== ======
 Information
======================== ======
 HTTP Method              POST
 Requires Authentication  Yes
======================== ======

Request Body
------------

.. code-block:: javascript

    {
        "data": [
            {
                "databox_id": 1,
                "record_id": 118
            }
        ]
    }

Response Fields
---------------

============= ================================
 Field         Description
============= ================================
 elements     The newly added basket elements
============= ================================
