Les fondamentaux
================

.. topic:: L'essentiel

    Cette section du manuel utilisateur liste les savoirs et fonctionnalités
    essentiels pour consulter un fonds sous *Phraseanet*. 
    Elle se destine aux **utilisateurs finaux** dont le but est la recherche et
    le téléchargement de media.
    
    Cliquer sur les liens proposés pour accéder à la présentation de ces actions
    essentielles.
