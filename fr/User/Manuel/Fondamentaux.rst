Les fondamentaux
================

.. topic:: L'essentiel

    Cette partie du manuel utilisateur regroupe les savoirs et fonctionnalités
    essentiels pour consulter un fonds dans Phraseanet. Elle se destine aux
    utilisateurs finaux dont le but est l'accès, la recherche et le
    téléchargement de médias dans l'interface Production.

    Cliquer sur les liens proposés pour accéder à la présentation de ces actions
    essentielles.
