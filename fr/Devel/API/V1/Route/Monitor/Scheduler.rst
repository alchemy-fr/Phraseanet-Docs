Monitorer le Scheduler
======================

About
-----

  Retourne des informations sur le statut du scheduler

  .. code-block:: bash

    /api/v1/monitor/scheduler/

  ======================== =====
   Informations
  ======================== =====
   HTTP Method              GET
   Requires Authentication  Yes
  ======================== =====

Parameters
----------

  Aucun paramètre

Response Fields
---------------

  =============== ======================================
   Field           Description
  =============== ======================================
   scheduler       Informations sur l'état du scheduler
  =============== ======================================
