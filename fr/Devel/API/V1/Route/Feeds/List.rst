Lister les flux
===============

A propos
--------

Liste tous les flux disponibles

.. code-block:: bash

    /api/v1/feeds/list/

========================== =====
 Informations
========================== =====
 Méthode HTTP               GET
 Authentification requise   Oui
========================== =====

Paramètres
----------

Aucun paramètre

Attribut de la réponse
----------------------

============= ================================
 Attribut      Description
============= ================================
 feeds         Les liste des flux
============= ================================
