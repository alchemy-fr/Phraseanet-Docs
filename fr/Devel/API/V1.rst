Documentation API v1
====================

L’API Phraseanet permet de développer des applications tierces exploitant les
contenus d’une médiathèque Phraseanet.

Pour partager vos retours d’expérences et publier vos codes sources,
utilisez nos :doc:`forums </Devel/Forums>`.

Pour démarrer :

* Lire l':doc:`introduction <V1/Introduction>`
* Se familiariser avec :doc:`oAuth2 <V1/Authentification>`
* Trouver la route :doc:`route <V1/Routes>` !

Vous rencontrez un bug ? Merci d’en rendre compte dans notre `mailing-list`_ ou
de créer un ticket dans notre `bug tracker`_.
